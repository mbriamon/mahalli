# Mahalee 🇯🇴 by Mary Briamonte(A) and Layann Wardeh(B)

A tourist spot discovery app for Jordan. Browse and search cultural landmarks, nature sites, museums, and more.

## Features
- Browse tourist spots across Jordan
- Search by name, category, and city
- Click on a spot to view more details
- Data stored and retrieved from Back4App (Parse)

## Tech Stack
- React + Vite
- React Router DOM
- Parse / Back4App

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173`
